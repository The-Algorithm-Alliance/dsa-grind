# 🥷 The Algorithm Alliance | DSA Grind

Welcome to the central repository for our DSA study group. This is our digital training ground to practice data structures, analyze time complexities, and keep each other accountable.

## 📌 The Ground Rules
1. **The Weekly Quota:** We solve **2–3 problems per week**. Topics change every Sunday.
2. **The Isolation Rule:** Only write, edit, or delete files inside your *own* personal workspace folder. Look at others' code, but don't touch it!
3. **Complexity Check:** Always try to include a comment at the top of your files with the Time and Space complexity (e.g., `// Time: O(N), Space: O(1)`).
